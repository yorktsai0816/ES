臺中文學館 // 臺中文學季
  • - 舉辦市集：好好手感微笑市集
  • - 舉辦時間：2015 / 09 / 01 (六)、9 / 02(日）
  • - 報名時間：即日起 ~ 7/29截止
  • - 舉辦地點：403 台中市西區樂群街48號
  • - 招募對象：以書籍、繪畫、各類紙品創作、紙膠帶、文具相關為優先。
  • - 所需費用：自帶桌NT600每攤 / 場次、 租借桌椅NT750每攤 / 場次(桌x1-120cmx60cm、椅子x2)
  • - 設備須知：不提供電力。提供 - 大傘（直徑230Cm)
  • - 招募攤位：50攤
  • - 活動簡介：投入響應文學季聯名推廣，支持所有藝文創作者能有更友善的展現舞台，入選手作品牌攤位將不予收取任何攤位費用。以書籍、繪畫、各類紙品創作、紙膠帶、文具相關為優先。
注意事項
  • 本市集有權依性質配置或臨時調動攤位位置。
  • 所有參加的品牌都必須有自己的小招牌以供識別。
  • 攤位嚴禁私下轉讓、合攤。
  • 當須配合主辦單位活動宣傳，分享活動海報或活動專頁連結，亦可自行設計海報宣傳。
  • 當日現場販售商品必須與報名表所填寫的項目相符。
  • 嚴禁使用擴音器材或自行撥放音樂，叫賣招攬生意以不影響其他攤友方式。
  • 謝絕仿冒品、批發品及侵權疑慮品以及宣稱療效品如有違反取消資格。
  • 如因天氣不可抗拒因素停辦，退費將會扣除場地、文宣費NT150。
  • 個人因素需請假於活動前7日提前告知者，如未提前告知，恕不退費。
  • 請假退費將會扣除銀行轉帳費用。
